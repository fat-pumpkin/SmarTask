## [1.0.5] - 2026-07-13

### 🐛 修复

- 修复移动端输入框太小导致输入内容被遮挡的问题，改为 textarea 支持根据内容自动展开
- 修复"添加"按钮出现在输入框内遮挡输入内容的问题，将按钮移至工具栏同行
- 修复移动端功能栏占用太多空间的问题，改为可折叠模式，移动端默认折叠
- 修复移动端标题与功能按钮分行显示导致空间浪费的问题，合并为同一行
- 修复看板/列表视图中操作按钮（编辑/添加子任务）重叠和挤压文字导致竖排的问题
- 修复看板模式下始终显示待办和已完成两列的问题，根据筛选状态显示对应列
- 修复交错时间线在移动端布局异常的问题，改为单列布局
- 修复代码中直接设置 style 属性的 lint 警告，改用 CSS class 和 setCssStyles
- 移除所有 CSS 中的 !important，改用更高特异性选择器

### ✨ 优化

- 操作按钮统一为 meta-action-btn 样式，移至任务描述下方 meta 行内水平排列
- 看板列在移动端改为纵向堆叠，避免水平滚动
- 卡片网格在移动端改为单列布局
- 日历格子触摸区域优化
